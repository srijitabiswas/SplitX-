# SplitX 💸

SplitX is a smart expense-tracking application designed to simplify group spending, settlements, and shared financial management. Whether you're traveling with friends, sharing rent with roommates, or managing team expenses, SplitX helps keep everyone on the same page.

## ✨ Features

- Create and manage expense groups
- Add and track shared expenses
- Automatic expense splitting among group members
- Settlement tracking between users
- User trust score and badge system
- Real-time balance calculations
- Clean and intuitive interface
- MongoDB-powered data storage

## 🛠️ Tech Stack

### Frontend
- HTML
- Tailwind CSS
- JavaScript

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
